
### Quick start


```bash
# clone
$ git clone https://github.com/brood915/MAP.git

# change directory
$ cd MAP

# install the dependencies with npm
$ npm install

# start the server
$ npm start
```

go to [http://localhost:8080](http://localhost:8080) in your browser.




Used https://github.com/preboot/angularjs-webpack as boilerplate.
